# EAN Barcode Scanner

Simple React Native app that scans EAN8 and EAN13 barcodes using Expo Camera.

## Setup

```bash
npm install
```

## Run

```bash
npx expo start
```

## Dependencies

- expo-camera
- react-native

## Use

1. Grant camera permission
2. Point camera at EAN8 or EAN13 barcode
3. Alert shows barcode data
